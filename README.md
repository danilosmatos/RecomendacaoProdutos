# RecomendacaoProdutos
## Projeto de Recomendação e Categorização de Produtos
### A nova shoppe da shoppe!

Este é um sistema de gerenciamento de estoque de livros com uma interface gráfica (Tkinter) usando Árvore AVL para organização e recomendação de categorias. O projeto segue o padrão MVC (Model-View-Controller) para separação de responsabilidades.

### Tecnologias Usadas
Python 3.10+ e Tkinter

### Estrutura do Projeto
* main.py: Incializa o **controller** e **view**, associa os dois.

* front_end.py: Usa o Tkinter na classe **ProductView** para fazer a interface gráfica.

* productController.py: Essencialmente o cérebro, usa o **Produto** para fazer a parte do backend para o **ProductView**, associa os dois com a árvore avl também.

* srph.py: responsável pela árvore AVL

* benchmark.py: Mede a performance da arvore avl

### Execução do Projeto

Baixar os arquivos e executar o main.py, nenhuma importação externa requerida.
